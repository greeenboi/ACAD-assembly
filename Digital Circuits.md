### Sequential Circuits
● Latch:
	○ Stores one bit of information i.e. memory element.
	○ Data is latched or stored until the next input is applied.
● Flipflop:
	○ Stores one bit of information. I.e. memory element.
	○ Data is stored on clock edge i.e. rising edge or	falling edge.
